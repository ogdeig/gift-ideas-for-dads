HOW TO DEPLOY ON GITHUB PAGES
-----------------------------
1) Create a new repo: https://github.com/new  (example: gift-ideas-for-dads)
2) Upload these files (or upload the ZIP then “Extract” locally before pushing).
3) In the repo: Settings → Pages → Source: select “main” branch and “/root”.
4) Wait ~1–2 minutes, then open:
   https://yourusername.github.io/gift-ideas-for-dads/
